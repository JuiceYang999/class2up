

## 创建您的前端项目：

  * 这是为了测试您的 Vue 环境是否已准备好创建项目。
  * `cd` 到您希望存放前端代码的目录（**注意**：这*不*应该在您的 Python 后端项目文件夹*内部*，而是在同一级别，例如 `Battery_Project/frontend` 和 `Battery_Project/backend`）。
  * 运行：
    ```bash
    vue create my-battery-dashboard
    ```
  * Vue CLI 会询问您选择哪个预设 (preset)。对于锂电池监测，选择 **"Manually select features" (手动选择功能)** 通常是最好的，确保至少选择 **Router** (用于页面导航) 和 **Babel**。
  * 创建完成后，按照提示的命令启动开发服务器：
    ```bash
    cd my-battery-dashboard
    npm run serve
    ```
  * 在浏览器中打开 `http://localhost:8080` (或提示的地址)，您应该能看到 Vue 的欢迎页面。
